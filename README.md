# 🛒 ShopEase - Django E-Commerce Website

A full-stack E-Commerce web application built using Django, Bootstrap, and SQLite.

---

## 📌 Features

- User Registration
- User Login & Logout
- Product Listing
- Product Details Page
- Shopping Cart
- Add to Cart
- Increase/Decrease Quantity
- Remove Products
- Checkout
- Order Processing
- Order Success Page
- Django Admin Panel

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/iltaf2003/CodeAlpha_ShopEase.git
```

Go to the project folder

```bash
cd CodeAlpha_ShopEase
```

Create Virtual Environment

```bash
python -m venv venv
```

Activate Virtual Environment

Windows

```bash
venv\Scripts\activate
```

Install Requirements

```bash
pip install -r requirements.txt
```

Run Migrations

```bash
python manage.py migrate
```

Start Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
